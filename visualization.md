# Data Visualization

- D3js

## Four steps of Data Visualization

    1. Transform Data
        - d3.cross
        - d3.max
    2. Map data to image space
        - d3.scaleLinear
        - d3.scaleTime
    3. Compute layout
        - d3.path
        - d3.treemap
    4. Draw the chart
        - d3.select
        - d3.append
